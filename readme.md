# LP Solar — Landing Page para Energia Solar

Landing page desenvolvida para captação de leads qualificados para empresas de energia solar.

O projeto foi criado com foco em:
- Conversão
- Performance
- Responsividade
- Integração com WhatsApp
- Estratégia de marketing digital

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla

---

## 📁 Estrutura do projeto

```bash
lp-solar/
│
├── imagem/
├── js/
│   └── script.js
│
├── index.html
├── style.css
├── README.md
└── .gitignore
```

---

## 🎯 Funcionalidades

- Landing page responsiva
- Integração com WhatsApp
- Formulário de captação
- Design otimizado para conversão
- Sessões estratégicas de marketing
- Estrutura voltada para tráfego pago

---

## 📲 Integração WhatsApp

O formulário envia automaticamente os dados preenchidos para o WhatsApp através da API oficial do WhatsApp.

---

## 🌐 Objetivo

Gerar oportunidades qualificadas para empresas e integradores de energia solar através de:
- Google Ads
- Meta Ads
- Landing Pages
- Estratégias de conversão
